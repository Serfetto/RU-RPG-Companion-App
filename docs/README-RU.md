# RPG Companion App — русский перевод

Этот репозиторий содержит **русский перевод существующего приложения [RPG Companion App](https://rpg-companion.app/)**.

Это **не отдельное приложение и не форк с новой функциональностью**, а набор переведённых файлов системы, ресурсов и вспомогательных скриптов, необходимых для локализации интерфейса и связанных игровых данных на русский язык.

## Что здесь находится

- перевод системных файлов приложения на русский язык;
- переведённые UI-строки, секции листа персонажа, характеристики, системные описания и другие ресурсы;
- вспомогательные скрипты для автоматизации перевода и обновления локализованных файлов.

## Цель проекта

Цель этого репозитория — адаптировать существующее приложение **[RPG Companion App](https://rpg-companion.app/)** для русскоязычных пользователей, сохранив совместимость со структурой оригинального проекта и его ресурсами.

## Важно

- Этот репозиторий основан на уже существующем проекте **[RPG Companion App](https://rpg-companion.app/)**.
- Основная задача здесь — **локализация на русский язык**, а не переработка логики приложения.
- Перевод **может быть не полностью точным** и в некоторых местах может требовать дополнительной ручной правки и проверки в интерфейсе приложения.
- Проект находится в процессе локализации и постепенного уточнения терминологии.


## Как выполнялся перевод

Весь перевод, а также написанные для него вспомогательные скрипты, создавались **с помощью нейросетей**.

Основная часть переводов бралась с сайта **[dnd.su](https://dnd.su)**, когда для соответствующих сущностей и текстов удавалось найти подходящие совпадения.

Если нужный текст не удавалось найти на **[dnd.su](https://dnd.su)**, он отправлялся в локально развернутый переводчик на базе LLM для перевода на русский язык.

Для автоматизации перевода использовалась локально развернутая модель: **[YandexGPT 5 Lite](https://huggingface.co/yandex/YandexGPT-5-Lite-8B-instruct-GGUF)**

Нейросети использовались как инструмент ускорения локализации, после чего результаты дополнительно проверялись и адаптировались под структуру файлов приложения.


## Статус

Проект находится в процессе локализации и доработки.

Возможны:
- неточности перевода;
- несоответствия терминов между разными частями системы;
- строки, которые ещё требуют ручной проверки в приложении;
- места, где автоматический перевод лучше заменить на более естественный вариант.

## Назначение репозитория

Этот репозиторий предназначен для:
- русификации существующего приложения;
- хранения локализованных файлов;
- хранения скриптов, использованных для перевода и последующего обновления локализации.
